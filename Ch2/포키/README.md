### 연습 문제 1

```java
class Practice1 {
    public static int[][] identityMatrix(int n) {
        int[][] result = new int[n][n];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (i == j) {
                    result[i][j] = 1;
                }
            }
        }
        return result;
    }
}
```

Q1. O(n^2)  
Q2. 그냥 for문 돌면서 index 2개 모두 i값을 넣으면 된다.
```java
class Practice1 {
    public static int[][] identityMatrix(int n) {
        int[][] result = new int[n][n];
        for (int i = 0; i < n; i++) {
            result[i][i] = 1;
        }
        return result;
    }
}
```
Q3. 그렇다. O(n^2) -> O(n)
